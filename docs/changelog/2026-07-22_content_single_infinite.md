**状态：** 已定稿并落地

## 1. 产品定义

| 项 | 内容 |
|----|------|
| 几何 | 永远同一套满槽 L0/L1/L2（坐标不变） |
| 变化 | seed → 点数 / 开局对 / 锁 / stock |
| 推进 | **无「下一关」**；胜利后 **再来一局** |
| 重开 | **重打本 seed**（同配点再挑战） |
| 新局 | 底栏 **新局** / 胜利 **再来一局** → 随机 seed，局号 +1 |
| 难度 D23 | 常规 **困难**（多锁、高 chain、薄库）；**每 3 局一极难**（第 3/6/9…：锁×3 + 近必 chain + 无 L1 伙伴灌库） |
| 路径锁 D24–26 | 钥匙稀缺 2–4；库钥匙靠前；残局 trim 多余库牌 |

顶栏示例：`第 3 局 · 极难 · 锁×3 · #18492011`
