## 1. 点选配对（选 A1 → 点 A2）

| 项 | 定稿 |
|----|------|
| 路径 | A1 **直线**飞到 A2，A2 **不动** |
| 曲线 | ease-in-out（非 easeOut 弹射、非粘滞 easeMeet） |
| 弧线 | **无**（`meetArcPx: 0`，产品否决飞入上抛弧） |
| 层级 | A1 ≈ 9500，避免从其它牌中间穿过 |
| 时长 | base 120ms，距离加成，上限 160ms |
| 缩放 | 飞行保持选中尺度，落地前 25% 再 matchPop |
| 之后 | **与拖到 A2 同一套 exit**（分抛/上抛/转/pop） |

---

## 2. 上抛角度 ↔ 飞入 / 拖入

| 路径 | 参考方向 |
|------|----------|
| 点消 | A1→A2 单位位移 |
| 拖消 | 松手前速度单位向量 |

在左右分飞基础上叠加 `exitApproachBias≈0.34` 的方向分量，并保底上抛。

## 3. 选中态

| 项 | 定稿 |
|----|------|
| 放大 | **1.1**，绕**牌心** |
| 谁放大 | **仅 selectedId**；意图伙伴 scale=1 |
| 晃动 | 极轻（±0.35/0.45 px，±0.55°），周期 **2s**，牌心 |
| 结束 | 拖起 / 配对 / 取消 → 停 idle |
