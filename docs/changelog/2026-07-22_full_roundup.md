## 0. 一句话现状

**单关无限 seed 的纸牌层叠配对**：固定满槽几何；同点同色（仅♥/♠）；清桌即胜；路径锁 + 钥匙稀缺；困难抬高且每 3 局一极难；库是工具（精简、钥匙靠前、残局回收多余）。

---

## 1. 产品规则定稿（玩家向）

| 主题 | 规则 |
|------|------|
| **胜利** | 谜题区清空即可（D10）；库/抽出叠不挡胜利 |
| **抽牌区** | 工具，不可直接点；只能抽到抽出叠再配（D10b） |
| **清桌** | 未用库牌 **全部回收** |
| **残局** | 桌上已不需要的同色同点，从库/抽出叠 **成对回收**（防「桌 1 张、库一摞」） |
| **配对** | **同点数且同花色**；红=仅♥，黑=仅♠（D22） |
| **内容** | 无多关战役；**单关无限**；再来一局=新 seed；重打=同 seed（D21） |
| **难度** | 常规 **困难**；第 **3/6/9…** 局 **极难**（D23） |
| **锁** | **路径锁**（L1 组顶瓶颈），非唯一钥匙状态机；同色同点即可开 |
| **钥匙密度** | 每把锁的同色同点全场 **2～4 张**（含锁）（D24） |
| **库内钥匙** | 若库中有钥匙，**尽量靠前**（早抽到、可被盖、洗回再现；不钉最后一张）（D25） |
| **平行剥** | 两组同顶 ⇒ 次顶必须不同（D19 / R15） |
| **层级** | L0 底 5×4×3；L1 中 4×3×3（锁在组顶）；L2 顶 3×2×2（开局 free） |

### 操作对照

| 按钮 / 手势 | 行为 |
|------|------|
| 再来一局 / 新局 | 新 seed；局号 +1；按局号选 hard/extreme |
| 重开 / 重打本 seed | 同 seed、同难度再开 |
| 撤销 | 同局回退 |
| 抽牌 | stock→waste；仅 waste 顶可配 |
| **点选** | 点两张同点同色 free → 消；不同 → 改选 |
| **拖放** | 拖 free 到同点同色松手 → 消；否则弹回（见 `drag_match_pile_shadow`） |

### 难度两档

| | 困难 hard | 极难 extreme（每 3 局） |
|--|-----------|------------------------|
| 锁 | 1～3（无 0 锁） | 固定 3 |
| 锁链 | 较高概率 chain | 近必 chain |
| 库伙伴 | 至多 1 组 L1 伙伴 | 0 组（更瘦） |
| 钥匙稀缺 | 2～4 / 锁 | 同 |
| 发局 | **必须**贪心可清（D27） | 优先可清；失败才密度兜底 |

## 5. 路径锁概念（笔记口径）

```text
路径锁 = L1 组顶瓶颈
       + 同色同点可开（非唯一钥匙状态机）
       + 全场该色点仅 2～4 张（稀缺）
       + 优先桌面路径；库里有则靠前
```

| 不是 | 是 |
|------|-----|
| RPG 唯一钥匙 | 遮挡路径 + 稀缺同色同点 |
| 锁特殊点击规则 | 与普通牌同一套配对规则 |
| 钥匙必须 L0 | 可在 L2/L1/L0/库；库则靠前 |

## 6. 层级关系（L0 / L1 / L2）

| 层 | 位置 | 规模 | 玩法角色 |
|----|------|------|----------|
| **L2** | 最高 | 3×2 组 ×2 张 | 开局 free；桌面挖掘链 |
| **L1** | 中 | 4×3 组 ×3 张 | **路径锁在组顶** |
| **L0** | 底 | 5×4 组 ×3 张 | 后段清场 |

更高层 footprint 挡住下层 → 不可点；局部揭开（压住的消了就能动）。
