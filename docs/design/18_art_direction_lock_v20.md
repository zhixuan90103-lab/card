## 2. 色板（落地建议）

| 角色 | 建议色（可微调） | 用途 |
|------|------------------|------|
| 背景 | `#F2F0EB`～`#F5F3EE` **铺满**，无第二层底板 | 整屏 |

| 黑（♠） | `#2C3338` 炭黑 | 点数/花色 |
| free 描边 | `#F0C4A8` 浅桃 1.5～2px | 可点提示 |
| 选中 | `#E8B84A` 柔金环 | 选中 |
| 底栏主按钮 | `#5B8DEF` 软蓝 | 抽牌/撤销 |
| 底栏次按钮 | `#9AA3AD` 灰 / `#6BAE8F` 绿 | 重开/新局 |
| 顶栏字 | `#8A939C` 次级灰 | 不抢牌桌 |
