# 07 · 术语表

**更新：** 2026-07-21  
**用途：** 统一用词，避免「金字塔 / 接龙 / 羊了个羊」混谈。

---

## 结构与几何

| 术语 | 定义 | 勿混淆 |
|------|------|--------|
| **谜题区** | 层叠牌阵所在区域，清完即胜利 | 不要叫「三角塔」 |
| **抽牌区 / stock** | 背面库；**工具**；清桌回收；残局 trim 多余 | 不可直接点；`stock[0]`=下一张 |
| **抽出叠 / waste** | 抽出后压栈，仅顶张可与谜题 free 配 | 可盖住先抽的钥匙 |
| **平行剥** | 两组同顶且同次顶的廉价同步展开 | 禁止（R15） |
| **路径锁** | L1 组顶瓶颈；同色同点可开 | 非唯一钥匙状态机 |
| **match-key** | 点数+颜色（如 2_red） | 配对与稀缺统计单位 |
| **钥匙（路径锁语境）** | 与锁同 match-key 的其它牌 | 全场该 key 仅 2～4 张 |
| **L0 / L1 / L2** | 底 / 中 / 顶板层 | 锁在 L1 顶；开局 free 在 L2 |
| **困难 / 极难** | hard；每 3 局 extreme | 顶栏可见 |
| **底层 / 底阵** | **4×5** 网格上的最下层牌（20 格） | 不是 7 行三角底 |
| **上层 / 盖层** | 盖在更低层之上的牌，制造遮挡 | |
| **层叠 / 遮盖** | 高层牌与低层牌占位重叠则压住 | 不是接龙「压序列」 |
| **自由 / 可选** | 无更高层重叠遮挡，且已翻开，可点击 | |
| **金字塔（本项目口语）** | **仅指**「往上越叠越少的层叠结构」比喻 | **≠** Pyramid Solitaire 三角布局 |
| **经典三角塔** | 1+2+…+n 行的 Pyramid Solitaire | **已否决**为本案主结构 |

## 抽牌

| 术语 | 定义 |
|------|------|
| **抽牌区 (Stock)** | 未抽出的牌堆，点此抽牌 |
| **抽出叠 (Waste)** | 已抽出的牌叠；**仅顶张可选** |
| **洗回** | 抽牌区空时，抽出叠整叠洗回抽牌区 |
| **随时可抽** | 不限制「谜题区有对时不能抽」 |

---

## 消除与操作

| 术语 | 定义 |
|------|------|
| **同点配对 / 同色同点** | 点数（rank）相同且 **同花色/同色** 的两张可消（**D22**：红仅♥、黑仅♠）；异色不可消 |
| **手动点选** | 先点 A 再点 B；系统不自动消 |
| **改选** | 第二张不同点时，选中改为第二张 |
| **收走 / 飞走** | 配对成功，两张离开牌面 |

## 对标品类（一句话）

| 名称 | 与本案关系 |
|------|------------|
| **羊了个羊** | 层叠遮挡参照；消除是 3 消进槽，我们是 2 点直消 |
| **接龙 / Solitaire / FreeCell** | **不做**列间接龙 |
| **Pyramid Solitaire** | 三角布局 + 常和 13；**布局已否决**，仅历史对照 |
| **Water / Parking / Screw** | 调研体验锚点，非本案直接抄规则 |
| **Goods 三消** | 另一 DNA，不做方法论硬套 |

## 工程 / 视口（D15–D17）

| 术语 | 定义 | 勿混淆 |
|------|------|--------|
| **设计分辨率** | 逻辑坐标 **393×852**（iPhone 15 CSS px） | 不是物理 1179×2556 |
| **phone-frame** | PC 上固定 393:852 的 contain 容器 + letterbox | 不是仅靠 DevTools |
| **letterbox** | 两侧或上下黑边，保证比例不裁切玩法区 | 不是 cover 裁切 |
| **逻辑 hit-test** | 先 `isFree` + AABB，再响应点击（D17） | 不是只靠 GPU/引擎顶层拾取 |
| **主渲染** | **PixiJS v8** 画牌桌；HUD 用 DOM | 不与 Three 双引擎混用 |

## 文档中废弃用语（写作时避免）

| 避免 | 改用 |
|------|------|
| 「三角金字塔」「1+2+3 塔」指本案 | 谜题区 / 4×5 层叠 |
| 「接龙一下」「搭上去」 | 配对消除 / 盖层遮挡 |

| 「清塔」易联想三角塔 | **清谜题区 / 清场** |
| 「用 Three 做 2D 牌桌就行」（默认） | 主引擎已定 **Pixi**（D15）；Three 非 MVP |
