# 01 · 愿景与产品定位

**状态：** 已对齐（2026-07-21）  
**范围：** 欧美手机 · 机制向 · 可堆关 · 体验优先（暂不展开商业化方案）

---

## 1. 产品一句话

> 纸牌题材的 **层叠遮挡配对谜题**（**4×5 底 + 上层遮盖**，类羊了个羊，**非**经典三角纸牌塔）：手动点两张同点数消除；可抽牌辅助；**清空谜题区过关**。

---

## 2. 体验主轴

| 层级 | 内容 |
|------|------|
| **主卖** | **顿悟（E5）**——卡住 → 想通先消哪对 / 何时抽 → 局面打开 |
| **辅卖** | 计划感（E4）、消除后的释放（E2） |
| **保护** | 安全探索（撤销/重开）、默认可解关卡、无默认 timer |
| **弱化** | 纯催眠清台、纯手气狂抽（用关卡结构托住规划） |

体验原则（调研共识）：

- 玩家买的是**感受**，不是机制分类号  
- 机制是运载工具；**感受曲线不同**才有体感差异  

---

## 3. 题材与规则边界

| 是 | 否 |
|----|-----|
| 纸牌 = **题材 / 美术语义**（点数、花色） | 纸牌 ≠ 接龙 Solitaire / 蜘蛛 / 空当 FreeCell |
| 规则 = **遮挡 + 同点配对收集** | 列与列之间 **不接龙搭龙** |
| 关卡制一关一布局 | 无限经典单局碰运气为主 |

**曾讨论后否决或降级：**

- 以 FreeCell 式「红黑递减叠列」为核心 → 与接龙无本质区别，已放弃  
- 「钥匙牌」作为与普通牌同规则的玩家卖点 → 逻辑弱，第一版不对玩家强调  
- 默认 timer、ads-or-lose 当难度 → 体验红线，禁止  

---

## 4. 与品类调研的关系

调研结论（摘要）：

- 千关机制战场在 **约束规划 / 调度 / 拆除** 等核；Goods 三消是另一 DNA  
- 非谈判：**默认可解信任、默认无 timer、关内少硬打断、目标一眼懂**  
- 创新宜从**体验**出发，再选载具  

本产品落点：

- 载具形态：**4×5 底层 + 分层遮盖 + 同点配对**（类羊了个羊层叠，非三角纸牌塔、非接龙）  
- 用纸牌皮承载，**不走接龙语法**  
- 用关卡结构与抽取节奏追求 **顿悟**，而非纯三消 dopamine  

详见：`research/sorting-market/08_round_synthesis_final.md`

---

## 5. 成功标准（体验向，暂定）

玩家能否用自己的话说出：

- 「要想一下先消哪对 / 什么时候抽」  
- 「消完会掀开一片」  
- 而不是：「就是接龙」或「就是无脑点抽牌」
