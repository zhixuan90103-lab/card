## 1. 行业共识（可读优先）

| 结论 | 来源倾向 | 对我们的含义 |
|------|----------|--------------|
| **大点数 / 大符号** | Senior Mahjong、商店卖点反复强调 large tiles / clear fonts | 牌面 52×72 内点数尽量大；花色清晰 |
| **可点与不可点必须一眼分** | free tile 高亮；covered 更暗/更扁 | free 抬起+提亮；背牌统一暗蓝灰 |
| **背景安静** | 成功作少花背景抢 tile | 深色或柔和单色，少纹理噪声 |
| **叠层用轻微 3D/阴影** | 经典上海/微软麻将、多数 3D tile | 不是纯扁平色块，要 **2.5D 厚度感** 才读得懂堆叠 |
| **主题可换，信息架构别换** | 多皮肤但布局固定 | 先定「信息清晰皮肤」，主题第二 |
| **底栏抽牌区不抢主舞台** | 标准 solitaire 分区 | 牌桌中上 70%，抽牌底 20% |

反面教训（上次效果图）：

- **纯瑞士扁平**：好看但不像「可玩牌桌」，叠层变糊  
- **赛博/霓虹**：风格强，伤扫读、偏主题包装  
- **绿绒赌场**：偏成人 solitaire，和「顿悟层叠」气质略远

## 2. 推荐主方向（本轮效果图按此出）

> **用户反馈（同日）：** 不要 2.5D/麻将瓷砖/赌场；要 **扁平 + 休闲感**（soft flat casual），非瑞士硬几何。

**名称 A（用户倾向）：柔和扁平休闲（Soft Flat Casual）**

| 层 | 建议 |
|----|------|
| 背景 | 浅暖灰 / 薄荷 / 淡紫灰，安静 |
| 背面 | **实心扁平**圆角色块 + 细边表示叠，无真实厚度光 |
| 正面 free | 奶油白，大号友好字体，红心黑桃 |
| free | 略亮 + **浅色描边**（非立体抬升） |
| 整体 | cozy puzzle，轻松，不赛博不赌场 |

**名称 B（调研备选，用户暂不倾向）：柔和 2.5D · 深色清扫读**

| 层 | 建议 |
|----|------|
| 背景 | 深灰蓝单色或极轻渐变，无强图案 |
| 背面 | 统一深蓝灰，圆角，**轻微厚度+软阴影** 表现叠 |
| 正面 free | 暖白/象牙，**大号点数**，红心/黑桃高对比 |
| free 状态 | 比背牌 **略放大或上浮** + 更亮 + 可选细描边 |
| 选中 | 金/柔光环，不挡点数 |
| 抽牌/抽出 | 分区清晰，抽出叠仅顶牌亮面 |
| HUD | 顶栏淡字；底栏扁平按钮但不抢牌 |

次选方向（各 1 张作对比）：

1. **暖纸感浅底**（治愈、仍 2.5D）  
2. **水墨/东方静物**（题材包装，信息架构同上）

| 现屏 | 调研后改什么 |

## 4. 效果图验收

- 不改变：手机框、L 层叠逻辑、抽牌/抽出叠位置、红♥黑♠  
- 必须能感到：哪几张能点、哪几张是盖着的  
- 像 **上架 puzzle**，不像海报插画或纯平面海报
