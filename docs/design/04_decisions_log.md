### D18 展开（2026-07-21 · 与 Level01 实现对齐，全文见 `05`）

| 条 | 内容 |
|----|------|
| D18a | **按层搭建**：L0 格心；L1+ 落在下层 **十字缝**；同层组间 **不互压** |
| D18b | 每组 **3 张**同尺寸；**上漏边**（底牌画面更靠上，组顶更靠下）；`d≈0.13H` |
| D18c | Free / 翻面 = **仅几何** `isCovering`（同组叠压 + 上层压缝）；**非整层锁死** |
| D18d | 压住本牌的上层组消完后，本牌可翻；别处上层不挡本地 |
| D18e | **牌面尺寸已定稿**：`CARD_W=52` `CARD_H=72`；同槽 `d=round(0.13×72)=9` |
| D18f | Level01 网格：**横 5 × 竖 4**；`GAP_X=8` `GAP_Y=28`；左右留白约 50.5 |
| D18g | Level01 递推：L0 5×4 → L1 4×3 → L2 3×2（**无 L3**；底网格/层数可随关改） |
| D18h | 消除仍为 **同点两张**；与 Disney ±1 接龙无关 |
