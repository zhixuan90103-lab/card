## 0. 一句话目标

> 在 **规则/配点已定稿、占位实现可玩** 的前提下，用 **有边界的检索 + 可填表的规格 + 短 POC**，为「能扫读的牌面、不误导的三态、清脆的消/翻反馈、干净的 HUD、真机不肉的操作感」收口，服务 **顿悟主轴**，而不是堆特效。

**产品滤镜（强制）：**

```text
主卖 = 顿悟（消序 / 抽不抽 / 掀哪条线）
纸牌 = 题材皮（点数花色可读），不是接龙拟真
输入 = 点选配对（非拖放对战、非三消拖拽）
引擎 = Pixi 8 + DOM HUD（禁止再开选型）
几何 = 393×852 · CARD 52×72 · free⇔亮面 · 逻辑 hit-test
```
