## 1. 概念框（定稿，检索滤镜）

### 1.1 路径锁定义

```text
路径锁 Path Lock
  = 高价值「想消但被几何挡住」的牌（现：L1 组顶）
  + 全场 ≥1 条可规划的同 match-key 获得途径（优先桌面）
  + 玩家须先打开压住路径，或规划抽序

不是：唯一钥匙 / 特殊点击 / 状态机解锁
```

| 层 | 内容 | 现状 |
|----|------|------|
| **L-geo** | 谁压住锁、旁路 | 满槽 isFree |
| **L-match** | 同色同点几张、在哪 | **密度未控（痛点）** |
| **L-tempo** | 何时撞锁 | L2 易 → L1 锁 |

### 1.2 路径锁 vs 唯一钥匙

见 `research/path-lock/01_problem_frame.md`。
