# 样板深挖：E1 Screw / Wood Nut & Bolt 系

**代表：** Screw Puzzle: Wood Nut & Bolt 及同核「拧螺丝 / 拆木板 / 螺母入孔」  
**方法：** 与 C1 同级的四层拆解（商店页 + 评论 + 既有社区检索；**非本机安装实机**）  
**分类自洽注：** 原 E「纯形位」已漂移；本品核近 **C 阻塞拆除 + 轻分类/容量**。战略上仍称 E1 锚点，叙事上可写作 **Screw-Jam（C 变体）**。  

**钉名参考：**  
- App Store: [Screw Puzzle: Wood Nut & Bolt](https://apps.apple.com/us/app/screw-puzzle-wood-nut-bolt/id6476347601)（Falcon Games）  
- 同核：Nuts & Bolts 3D、Wood Screw、Unscrew Master 等

## 四层拆解

| 层 | 内容 |
|----|------|
| **① 主题皮** | 木板层、彩色螺丝、螺母/孔位、工具感 UI；常加昆虫/松鼠等装饰障碍 |
| **② 现实脚本 L** | 拧螺丝、拆装配、修木工 — **弱劳动熟悉感**（非真实力矩/工具技能） |
| **③ 认知核** | **主 C7 阻塞拆除**（谁压在谁上面、先拧哪颗）；**辅 C5 分色入孔/盒** + 常有 **C4 容量**（孔/托盘位有限） |
| **④ 情绪兑现 E** | E4 计划、E5 顿悟、E8 拧下确认、E2 木板掉落/清层释放；浅关可有 E10 ASMR 拧感 |

### 真 / 半 / 假：**半真**

| 真 | 被替换 |
|----|--------|
| 层叠遮挡、先拆后卸的直觉 | 真实拧矩、工具操作、物理装配 |
| 「拆开/修好」的释放脚本 | 变成依赖序 + 分色容量 puzzle |

**玩家自我叙述：**「我在拧螺丝 / 拆木头。」  
**认知核复述：**「我在算可拆集合与顺序，并把同色钉归到有限孔位。」

→ 上瘾解释跟 **C7+C5 与 E4/E5/E2**；不要跟「模拟木工真实感」。

## 核心循环（单关 · 综合同核描述）

```text
观察木板层叠与螺丝颜色
  → 识别当前可合法拧下的螺丝（无被挡 / 规则允许）
  → 点选拧下（局部反馈：拧转动画 / 弹出）
  → 若有目标孔/盒：放入同色位（容量约束）
  → 木板失稳掉落 / 层清空（局部大反馈）
  → 重复直至清关
失败常见：孔位塞满、误序导致无解、（产品侧）timer 到 / 无道具
```

**社区策略共识（增强核判定）：**  
优先处理挡层 / 复杂区；把螺丝挪到 **empty holes** 腾位；保留空位别过早堵死；分色入孔 — 与 **C7 + C4/C5** 一致（类似 A 的「保护空管」）。

**与 C1 Parking / A1 Water 对照**

| | Screw | Parking | Water |
|--|-------|---------|-------|
| 主核 | C7 + C5 | C1 + C7 | C2 + C3 |
| 局部爽 | 拧下、掉板 | 出车 | 满色 |
| 工作区 | 孔位/托盘容量 | 出路与可动车 | 空管 |
| 皮 | 半真工具 | 半真车 | 假皮液体 |
| 千关 | 层数/颜色/挡位加密 | 车数/挡路深度 | 管数/层序 |
| 市场污染 | timer + ads-or-lose **极重** | 道具墙常见 | extra tube / 广告 |

## 上瘾结构

| 阶段 | 机制 | 状态 |
|------|------|------|
| 入局 | 一颗钉/一层板，秒懂 | E11 |
| 中段 | 可拧集合缩小，要规划孔位 | E4 |
| 卡点 | 孔满 / 关键钉被压 | E3 近失 |
| 击中 | 发现先拧哪颗 | E5 |
| 收尾 | 连续掉板 / 清层 | E2 |
| 再来 | 同规则加密 | 长关号社区存在 |

商店评论侧（Falcon 系）：**「关卡本身 fun」** 与 **「ads 比游玩时间长」** 同帖并存 → 核有效，产品污染极重。

## 舒适结构（核 vs 产品）

| 核可提供的舒适 | 市场上常被毁掉的方式 |
|----------------|----------------------|
| 规则短、点一下就有反馈 | 强制 timer（评论点名 dealbreaker） |
| 拧下/掉落的感官反馈 | 每关 2–8 分钟广告链 |
| 短关碎片 | 故意过难逼 booster/ad |
| 半真皮秒懂 | 克隆同质、无新意 |

**设计启示：** Screw 结构 **适合做无 timer 的放松向约束规划**；当前头部样本 **不能当「舒适产品」对标**，只能当 **机制核对标**。

## 流失 / 怒点（体验向 · 强证据）

| 点 | 证据强度 | 归类 |
|----|----------|------|
| 广告时长 > 游玩 | App Store 多条长评 | H10 干扰 |
| 故意难 + 道具/广告过关 | 「intentionally impossible」「1/10 无需 boost」 | H12 可解信任 |

## 手感要点（观察级，非本机）

| 项 | 推断 | 设计建议 |
|----|------|----------|
| 输入 | 点选螺丝为主 | 可拧高亮；非法点轻抖 |
| 反馈 | 拧转 + 弹出 + 掉板应分层 | 掉板 > 单钉拧下 > 普通点 |
| 3D | 层叠遮挡导致点错/看不清 | 半透上层、爆炸视图、可拧描边 |
| 孔位 | 拖或点入孔 | 强磁吸 + 容量数字 |
| 撤销 | 深关刚需 | 步撤优先于重看广告重开 |

---

## 设计启示（若做 Screw 结构 / C 变体）

1. **难度单位** = 层叠深度 × 颜色数 × 孔位容量 × 岔路，不是更仿真的螺丝刀。  
2. **默认无 timer** — 样本评论证明 timer 直接杀死本可成立的放松定位。  
3. **默认可解、无 ads-or-lose** — 否则核再好也变「恨」。  
4. **教程教「先看谁压住谁 + 孔位别堵死」**，不教拧螺丝手艺。  
5. **与 A/C 差异化：** 比 A 多重力学掉落反馈；比 C 多颜色归位层；勿再叠强制 PvP/八个副作用（样本版本史充满 meta 扩张）。  
6. **分类：** 对内可标 **Screw-Jam**，避免与纯「形位填孔 E」混淆。
