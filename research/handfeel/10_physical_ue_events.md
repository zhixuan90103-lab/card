## 总则

| 原则 | 内容 |
|------|------|
| 成功峰值同级 | 点消 ≡ 拖消 → 同 **P-meet + P-exit** |
| 意图辅助 | 仅 **free + 可配** 轻放大；非 free 不亮 |
| 抽两段 | 先位移到位 → 再翻 |
| 洗回两段 | 先扣背 → 再依次回库（N 大加速） |
| 非法不重罚 | 改选 / 弹回 / 挡牌静默 |
| 剂量 | 选中/高亮薄；消最厚；抽中；洗回中+加速 |
| 锁输入 | 消整段 busy；翻默认不锁；洗回建议短锁 |

### 剂量阶梯（修订）

```text
L0  挡牌静默
L1  选中浮动 · 改选 · 意图高亮
L2  拖跟手 · 非法弹回 · 抽移
L3  对撞+抛物线出场（成功峰值）
L3b 抽后翻 / 新 free 翻（结构兑现，弱于 L3）
L2b 洗回扣背+回库（批量加速）
L5  清场（仍 ≤1s 可点 CTA）
```

### 成功消合成路径

```text
点消: P-sel(A) → P-hint → 点B → P-meet(120) → P-exit 共抛(260) → P-flip(newly free)
拖消: P-drag(A) → P-hint → [P-aim] → 松手
        → 若已大致重叠: 直接 P-exit 共抛
        → 否则短 P-meet 再 P-exit
总 busy: ≤400ms 目标 / 硬顶 450
逻辑: match commit 即时；表现层 tween；busy 中禁撤/抽/点（E09）
S3: exit 前 skip pair、禁止全量 refresh；flip 在 exit 后
```

### 反查补钉（`13`）

| 项 | 钉 |
|----|-----|
| 双牌 exit | **共抛同步**（默认） |
| 意图高亮过多 | 最多 **4** 张 free 同 key；超出取最近 |
| 连消 | POC 默认 busy 全锁；P1 再谈跳过 |
| 洗回/出场中撤销 | **禁用** |
