## 1. 翻牌 + 缩放呼吸（PF5）

| 阶段 | 时间占比 | 动作 |
|------|----------|------|
| 收窄 | 0–45% | scale.x 1→0.05；**整体 scale 1→1.08**（呼吸起） |
| 换面 | ~50% | 背→面（或面→背扣牌） |
| 展开 | 50–100% | scale.x 0.05→1；**整体 scale 1.08→1** |

| 参数 | 推荐 |
|------|------|
| 总时长 | **160–200ms**（新 free）；抽后翻可 **140–180** |
| 呼吸峰 | **1.06–1.10**（默认 1.08） |
| busy | **否** |
| squash | 关 |

## 2. 抽牌：位移 → 翻（PF6 / E07）

```text
stock 顶
  → P-draw-move：飞到 waste 座位（120–180ms，ease-out，弧可浅）
  → 到位
  → P-draw-flip：伪翻+呼吸（140–180ms，不 busy 或仅 move 短锁）
```

| 规则 | 内容 |
|------|------|
| 串行 | **先到位再翻**（产品钉） |
| 空 waste 座位 | 飞向 waste 锚点 |
| 与 E06 | 同技法家族；抽翻可略短 |
| 旧 H-draw=0 | **废止瞬时默认**；改为上表 |

## 3. 洗回：扣背 → 依次回库（PF7 / E08）

```text
waste 可见牌（及叠内需表现的牌）
  → P-rec-back：全部/自顶向下 扣成牌背（并行总 ≤150ms 或 stagger 短）
  → P-rec-return：自顶（或自底）依次飞向 stock 座位叠放
       间隔 base=40ms；N>8 → 20ms；N>16 → 成组（每组 2–3 张）或剩余瞬移
  → 总时长硬顶建议 ≤900ms（理想 ≤700ms）
```

| 反模式 | |
|--------|--|
| 每张 200ms×20 张 | 拖死 |
| 无扣背直接瞬切回库 | 无「收起」感 |
| 回库中可乱点开新操作 | 逻辑乱（建议短锁） |

## 4. 消出场：对撞后抛物线（PF3）

接 B 轮汇合后：

```text
t=0 汇合完成（两牌重叠或紧贴）
  → 共同或各自：
     初速度上 + 轻微水平外抛
     旋转 ±15°～±45°（默认 ±30° 量级）
     重力感加速度（ease：先快后可加速下落）
     α 1→0
  → 出屏或 α=0 回收
时长 220–320ms
```

| 与旧 flyAway | |
|--------------|--|
| 旧 | 上浮+缩+淡 280ms |
| 新 | 汇合 100–160 + 抛物线 220–320 → **总长须控** |
| 建议 | 总成功演出 **≤400ms** busy；超时优先砍弧高/旋角而非砍确认感 |

点消 / 拖消：**同一 P-exit**（峰值同级）。

## 5. Pixi 实现笔记（PF9 · 短）

1. 路径：二次贝塞尔或 `x=lerp, y=y0+v*t+0.5*a*t²`  
2. 旋转：`rotation += ω * dt`（有上限）  
3. 翻：pivot 中心 + scale.x；呼吸用 uniform scale 叠乘  
4. zIndex：出场/拖中最高；回库按叠序  
5. 多张：对象池/复用 view，避免每帧 new  
6. 洗回 N 大：跳过不可见张的逐帧，只动顶 K 张
