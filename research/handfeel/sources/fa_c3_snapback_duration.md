# FA-C3 · 弹回时长与「纠正感」（Round 3）

**日期：** 2026-07-22  
**轮次：** 第三轮检索 · FA5 / A-snap  
**可信度：** 中（NN 100ms 重排 + 原则；仍无精确竞品 snap ms）  
**填入：** A-snap 区间 · C3

## 源

| # | 源 | 要点 |
|---|-----|------|
| 1 | [NN/g DnD](https://www.nngroup.com/articles/drag-drop/) | 松手后物体归位/让位动画约 **100ms** 量级，给物理操纵感；过慢像糊、过快像瞬切 |
| 2 | 既有 fa_a5 | 弹回=纠正非惩罚；短于 match；无红闪 |
| 3 | Articulate 等教学 DnD | 错项 snap-back 是标准 soft-fail；强调 **回原位** 语义，少报具体 ms |

## 观察

| 项 | 推论 |
|----|------|
| 下界 | ~**100–120ms** 仍可读（NN 重排带） |
| 舒适带 | **140–200ms**（既有区间）仍成立 |
| 与 match 关系 | **必须明显短于** match（280），否则错误像「半套成功」 |
| 180 默认 | 落在舒适带中上；A/B 试 **140** 合理 |
| 仍缺 | 逐帧竞品库 — **不补外搜** |

## 对本产品

| 决策 | Round 3 |
|------|---------|
| A-snap 默认 | **维持 180** |
| A/B | 140 vs 180 |
| 相对 match | snap ≤ ~0.65× match（180/280≈0.64）作纪律参考 |
| 红闪/重音 | 仍禁止 |

**弹回 ms 外搜封口；定稿归 C3。**
