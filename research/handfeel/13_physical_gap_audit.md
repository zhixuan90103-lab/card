## 3. 关键不自洽 / 空洞

### 3.1 🔴 算术：meet+exit 默认 vs A-clear

| 项 | 值 |
|----|-----|
| 默认 meet | 130 |
| 默认 exit | 280 |
| 加总 | **410** |
| A-clear 目标 | **≤400** |
| 硬顶 | 450 |

**后果：** 按默认常量实现会 **直接违反** 自己写的总长目标。  
**补钉（计划 v1.1 采纳）：**

```text
定稿默认（检索修订，仍为假设）:
  meetMs = 120
  exitMs = 260
  sum = 380 ≤ 400
A/B: (120+260) vs (130+280) 仅当总长≤450
```

### 3.2 🔴 双牌 P-exit 运动学未钉

| 未答 | 风险 |
|------|------|
| 两牌 **共用一个刚体** 抛出 vs **各自** 抛物线 | 实现分叉、观感不同 |
| 拖消时目标牌是否先 **瞬移汇合** 再抛 | 峰值是否仍「同级」 |
| 旋转同向/反向 | 视觉噪声 |

**补钉意向：** 默认 **汇合后绑定为 pair 容器共抛**（或同步同参双抛）；拖消 **短 meet 或 skip meet** 已有，须写死一种。

### 3.3 🟡 连消 / 输入队列

厚动画（~400ms busy）下连点下一对：

- 旧轨：busy 全锁  
- 新轨：未写 **队列 1 手** 或 **跳过剩余 exit**

**不补后果：** 高手手感肉。  
**补钉：** POC 默认全锁；P1 评估「exit 后半可被下一合法选中取消剩余 α」。

### 3.4 🟡 撤销 vs 长动画

P-meet/P-exit/洗回中途点撤销：未定义。  
**补钉：** busy 中撤销 **禁用**（与旧 E09 一致）；洗回中禁用抽/点。

### 3.5 🟡 意图高亮与多层 free 同 key>2

3+ 张 free 同 key：全放大是否吵。  
**补钉：** 最多高亮 **K=4**；超出只高亮几何最近。

### 3.6 🟡 抽/洗回与 session API

检索未对接：

- `draw` 是否异步等动画结束再改 waste 逻辑  
- 洗回是否 **逻辑先 commit 再播** 或 **播完再 commit**

**补钉：** 推荐 **逻辑即时 commit + 表现层 tween**（与消牌一致）；动画中 isBusy 挡输入。

### 3.7 🟡 旧债未并入本轨执行序

`08_self_consistency`：match 前 refresh 冲动画、无 freeBefore——**新抛物线若仍 refresh 会更惨**。  
**补钉：** POC **强制** S3：skip pair + 勿全量 refresh + flip 在 exit 后。

### 3.8 🟢 已够用

- 非法不重罚 / E05  
- 非 free 不高亮  
- 无默认震屏 / 无 squash  
- 抽串行、洗回两段结构  
- 生存手册迁移边界
