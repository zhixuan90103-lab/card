# 参数卡 · 牌面可读与无障碍对比

- 品类：实体/数字纸牌可读
- 输入：扫视识别 rank/suit
- 可信度：**中–高**（无障碍实践 + WCAG 对比原则）
- 源：
  - [Accessible Playing Cards For Low Vision](https://veroniiiica.com/playing-cards-and-low-vision/)
  - [WebAIM · WCAG contrast](https://webaim.org/articles/contrast/)
  - 行业常识：角标（corner index）双端镜像

| 项 | 观察值 | 备注 |
|----|--------|------|
| 布局 | **角标 index**（左上±右下镜像）+ 可选中心花色 | 小尺寸比单居中串更耐遮挡 |
| 红 | 宜 **偏深**；纯红 `#f00` 对白约 4:1，大字临界 | 现状 `0xc0392b` 可用；可试更深 `0xb71c1c` |
| 黑 | 近黑 on 浅底 | 现状 `0x1a1a1a` OK |
| 对比 | 正文字 AA **4.5:1**；大字 3:1 | 牌上 18px 按「大字」偏紧，仍宜 ≥4.5 |
| 色盲 | 红黑外 **花色符号** 必须独立承载信息 | 本产品仅 ♥/♠，符号差异大 |
| 轮廓 | 色块可加 **深描边** 增强小标 | 可选 |
| 选中 | 高对比描边（金/青）+ 位移 | 不靠变色 alone |
| 背面 | 低饱和、统一、**低于** free 亮度 | 不抢扫读 |
| 消牌/翻面 | — | — |
| HUD | 状态字勿与牌面抢对比 | 次要灰蓝 OK |

**可迁移：**  
- **双角标** rank+suit（字号 ~11–13）+ 中心大花色或大 rank  
- free 底保持高亮浅色；字用深色  
- 红略加深若真机发灰  

**不可迁移：**  
- 四色全副花色方案（产品只有 ♥/♠）  
- 大字全屏无障碍模式（P2）  

**对本产品：** 52×72 推荐 **角标优先**；中心可放花色 glyph 放大。
