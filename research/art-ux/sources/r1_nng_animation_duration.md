# 参数卡 · NN/g 动画时长（UI 微交互）

- 品类：通用 UX / 微交互（非纸牌，**时长权威**）
- 输入：点击反馈
- 可信度：**高**（Nielsen Norman Group 方法文）
- 源：[Executing UX Animations: Duration and Motion](https://www.nngroup.com/articles/animation-duration/)（2020；检索 2026-07-22）

| 项 | 观察值 | 备注 |
|----|--------|------|
| 简单反馈（勾选/开关态） | **~100ms** | 体感立即、可感知 |
| 一般动画区间 | **100–500ms** | 复杂度与位移决定 |
| 较大 UI 变化（modal 等） | **200–300ms** | 移动端合适 |
| 偏慢上限 | **~400ms** | 仅大位移；再长易烦 |
| 选中反馈 | ≤100ms 出态 | 映射 E01 |
| 消牌时长 | 200–300ms 带 | 映射 E03（非 1s+） |
| 翻面/露出 | 100–200ms | 映射 E06 |
| 过关到可继续 | ≤1s 可点 CTA | 体验合同；非 NN 原文 |
| free 可读 | — | 不适用 |
| 非法反馈 | 薄、短 | 与「不重罚」一致 |
| HUD 密度 | — | 不适用 |

**可迁移：**  
- 选中/取消 **≤100ms**  
- 消牌 **200–280ms**（现状 280 合格）  
- 翻面 **120–200ms**  
- 忌 >400ms 的常用动作  

**不可迁移：**  
- 网页表单审美直接套牌桌装饰  

**对本产品假设：** 消牌保持 240–280；翻面新建 160ms 默认。
