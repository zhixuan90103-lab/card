## 总则

| 原则 | 内容 |
|------|------|
| 合法立即反馈 | E01 / E-drag / E03a·b 必须可见 |
| 成功双路径同峰值 | **E03a ≡ E03b**：同 `flyAway`、同 L3；禁止点消瞬删、拖消加料分叉 |
| 非法不重罚 | E04 / E04b / E05 无震屏、无红闪扣分 |
| 剂量 | 常用薄、合法消中、清桌略厚（L0–L5） |

| Free=亮面 | 表现不得与 `isFree` 分叉 |
| 先消后掀 | E03 完成 → E06；见 `08` S3 |


## 文案定稿建议

| 场景 | 文案 |
|------|------|
| 默认 status | `同色同点配对 · 盖住的点不到 · 清桌即胜` |
| 软 tip | `试试撤销 · 或继续抽牌` |
| 硬死副 | `没有可配对的牌了` / `可撤销、重打本局，或开新局` |
| 胜副 | `再来一局换新配点 · 重开可打同一 seed` |
| 极难（顶栏已有） | 不另弹；勿说「唯一钥匙」 |
