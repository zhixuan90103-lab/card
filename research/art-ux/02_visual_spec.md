# R2 · 视觉规格 v0.3（经典角标牌面）

**状态：** **已落地** `theme.ts` · `cards.ts`（正/背对齐参考图）  
**约束：** CARD 52×72 · d=9 · Free⇔亮面 · 仅♥/♠ · 程序化 Graphics/Text  
**来源：** 参考图 — 乳白面 + 雾蓝水印背 + 角标 rank/suit + 中心大点数

## 1. 牌面正面（free）

| 元素 | 规格 | 代码常量 |
|------|------|----------|
| 底色 | `#FFF8F0` 暖象牙 | `Theme.face` |
| 描边 | `#E0D2BC`；选中金 `#E8B44A` | faceStroke* |
| 圆角 | **8** | RADIUS |
| 角标 | 左上/右下：`rank` 上 + 花色下 | 经典 index |
| 中心 | **大 rank** + 上下小花色 pip | 参考 K♥ |
| 红 / 黑 | `#C62828` / `#1C1C1C` | ink* |
| 阴影 | 双层 soft | shadow* |

**布局 ASCII（52×72）：**

```text
┌──────────────┐
│ K            │  角 rank
│ ♥            │  角 suit
│      ♥       │  中心上 pip
│      K       │  中心大 rank
│      ♥       │  中心下 pip（倒置）
│            ♥ │
│            K │  右下镜像
└──────────────┘
```

**禁止：** 非 free 画亮面字；四色花色；依赖纹理才能读 rank。

## 3. 三态（必须可辨）

| 态 | 视觉 | 逻辑绑定 |
|----|------|----------|
| **Free 亮** | §1 正面；无常驻外发光 | `isFree === true` |
| **非 Free 背** | §2 雾蓝背 | `isFree === false` |
| **选中** | 金描边 + **y -= 4** + zIndex 抬高 | `selectedId` |

**验收：** 5 秒内能指出全部 free（白面跳出蓝海）。
