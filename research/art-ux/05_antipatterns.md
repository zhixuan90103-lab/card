---

## 禁止（🚫）

| ID | 反模式 | 原因 |
|----|--------|------|
| X01 | 非 free 画亮面 / free 画背面 | 破坏 I5；误导可点 |
| X02 | 渲染 hit 与逻辑 isFree 不一致 | 破坏 D17 |
| X03 | 默认 timer / 倒计时 UI | 体验雷区 |
| X04 | 强震屏作反馈或惩罚 | 伤扫视；无障碍差 |
| X05 | 非法操作重罚动画（闪红扣分） | 合同：改选不重罚 |
| X06 | 消牌/过关演出 **>1s** 锁死且不可点 CTA | 拖节奏；社区高怒 |
| X07 | 文案写「唯一钥匙」「接龙」「进槽三消」 | 规则口径错误 |
| X08 | 为好看改 cover / 5×4 骨架 / 暗改 CARD 尺寸 | 须先改 `05` |
| X09 | 再引入 Three / 真 3D 翻牌主路径 | D15 |
| X10 | 背面高饱和大花纹抢 free 扫读 | 破坏 5 秒可读 |
| X11 | 红黑只靠色、无花色符号 | 色觉与小尺寸风险 |
| X12 | 关内硬广 / 失败强制看广告 | 雷区（本阶段） |

## 慎用（⚠️）

| ID | 项 | 条件 |
|----|-----|------|
| W01 | 粒子 / 全屏闪 | 仅 E12 极轻；E03 最多 1 帧 |
| W02 | 选中大幅旋转/倾斜 | 挡邻牌扫视 |
| W03 | 抽牌长轨迹 | 易抢消牌 juice；≤180ms |
| W04 | 命中盒远大于视觉 | 可轻微 pad；过大误点挡牌 |
| W05 | 教学句常驻两行 | 与顶栏抢；可缩或仅首局 |
| W06 | busy 过宽（翻面也锁） | 连点手感变肉 |
| W07 | 图集大图 | 包体；本轮优先程序化 |
| W08 | 「重打本 seed」过长按钮文案 | 可「重开本局」 |

## 推荐正模式（✅ 对照）

| 场景 | 正模式 |
|------|--------|
| 选中 | 短、金边、微抬 |
| 消 | 240–300ms 飞淡（H-match 默认 280） |
| 掀开 | 160ms 伪翻 |
| 非法 | 静默或改选 |
| 死局 | 软 tip → 硬浮层；CTA 明确 |
| 胜利 | 短文案 + 立刻可「再来一局」 |
