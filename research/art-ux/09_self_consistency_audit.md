### 3.2 🔴 代码路径 · `main` refresh 与 S3 / flyAway

**应然（08 S3）：**

```text
sync(skip=pair) → flyAway → onDone → flip → full sync
禁止 fly 中途全量 sync 冲掉动画
```

**实然（`main.ts` 匹配成功）：**

```text
cards.sync(state, pair)
refresh()           // 内含 cards.sync(state) 无 skip
                    // → alive=false → visible=false
cards.flyAway(pair) // 可能在不可见精灵上播
```

**后果：** 与「消牌兑现感」目标打架；基线 flyAway **可能已被 refresh 短路**。  
**R4 必改（实现，非再检索）：**

```text
refreshHudOnly 或 refresh({ skipCardIds: pair })
flyAway onDone 再 full refresh + flip(toFlip)
```

**判定：** 规格自洽，**规格↔代码不自洽**；属 R4 第一刀工程债。
